End Point Descreption Example
/addBook It add the data into the database. http://localhost:3000/addBook?bName=guns and thighs&bAuthor=rgv&releaseDate=2020-04-10&bPrice=100&bQuantity=12
/getOneBook It fetch the single book data based on book id. http://localhost:3000/getOneBook?bId=1
/getAllBooks It fetch all the books from database. http://localhost:3000/getAllBooks
/getAllBooksAescOnQuantity It fetch all the books in aesc order based on quantity from database. http://localhost:3000/getAllBooksAescOnQuantity
/getAllBooksAescOnName It fetch all the books in aesc order based on name from database. http://localhost:3000/getAllBooksAescOnName
/updateBook It update the book quantity based on the book id. http://localhost:3000/updateBook?bId=1&bQuantity=200
/deleteBook It delete the book based on the book id. http://localhost:3000/deleteBook?bId=1
